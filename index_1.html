Snippet 01 — Python · Detección geoespacial

Haversine para fraude de pagos

Si la distancia entre punto de inicio y fin del viaje supera el umbral del país, se marca como fraude; si está dentro, es mala práctica.

from math import radians, sin, cos, asin, sqrt def haversine_km(lat1, lon1, lat2, lon2): R = 6371.0 # radio terrestre en km dlat = radians(lat2 - lat1) dlon = radians(lon2 - lon1) a = sin(dlat/2)**2 + cos(radians(lat1)) \ * cos(radians(lat2)) * sin(dlon/2)**2 return 2 * R * asin(sqrt(a)) def clasificar(viaje, radio_km): d = haversine_km(viaje.lat_ini, viaje.lon_ini, viaje.lat_fin, viaje.lon_fin) return "fraude" if d > radio_km else "mala_practica"
Snippet 02 — SQL · ClickHouse

Deduplicación sin DISTINCT ON

ClickHouse no soporta DISTINCT ON al estilo Postgres. Patrón equivalente con GROUP BY + argMax() para tomar el registro más reciente por usuario.

-- último estado conocido por pasajero SELECT passenger_id, argMax(status, updated_at) AS last_status, max(updated_at) AS last_seen, count() AS events FROM picapmongoprod.passenger_events WHERE updated_at >= now() - INTERVAL 90 DAY GROUP BY passenger_id HAVING events >= 3 ORDER BY last_seen DESC;

from openpyxl import Workbook from openpyxl.styles import PatternFill wb = Workbook() ws = wb.active ws.title = "Alertas" ws.append(["piloto_id", "score", "severidad"]) for a in alertas: fill = PatternFill("solid", fgColor= "C0392B" if a.score >= 0.85 else "E67E22" if a.score >= 0.65 else "F1C40F") row = ws.max_row + 1 ws.append([a.piloto_id, a.score, a.severidad]) for col in "ABC": ws[f"{col}{row}"].fill = fill wb.save("alertas_fraude.xlsx")
Snippet 04 — Python · LLM local

Clasificación de mensajes con Llama 3.1

Llamada a Ollama corriendo local. Sin costos de API, sin enviar datos sensibles a terceros. Procesa 10K+ mensajes/mes.

import requests, json PROMPT = """Clasifica el siguiente mensaje como: - 'abusivo' : lenguaje agresivo o insultante - 'inapropiado' : solicitación sexual/comercial - 'normal' : conversación regular Responde solo con la etiqueta. Mensaje: {texto}""" def clasificar(texto: str) -> str: r = requests.post( "http://localhost:11434/api/generate", json={ "model": "llama3.1", "prompt": PROMPT.format(texto=texto), "stream": False, }, timeout=30, ) return r.json()["response"].strip().lower()
